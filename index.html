It appears that the quick search does not allow for multiple search parameters, such as finding a keyword and the teams with a github proposal. Also note, that the routes include /awards to capture the winning proposals.

Advanced Search

Several general search engines can be queried with advanced parameters, such as Google, DuckDuckGo, Bing.

Example advanced queries on Google
Find winner in the 2021 challenge with a Github solution. Note that winner is singular because that is how each winning team is listed.   
site:https://2021.spaceappschallenge.org "winner" "github.com"
For more results find nominee in the 2021 challenge with a Github solution.
site:https://2021.spaceappschallenge.org "nominee" "github.com"
For the most results, find all entries in the 2021 challenge with a Github solution.
site:https://2021.spaceappschallenge.org "github.com"
Or maybe, find all winners in the 2021 challenge with a YouTube presentation.
site:https://2021.spaceappschallenge.org "winner" "youtube" OR "youtu.be"

It is curious that the Google quick search can query across multiple url's with the help of the cse, and the advanced search can search in depth on only one url. Note that in the examples above the path does not have the /awards route which allows for a broader search refined by additional keywords.
